# Kairis Logo Concepts

These are rough first-pass SVG concepts for the `Kairis` brand. They are intended to test symbolism and direction, not to serve as finished logo assets.

## Concepts

- `kairis-threshold-k.svg`
  - Monogram-led concept
  - Represents a `K` crossing a threshold or execution gate
  - Strongest option for product iconography and app branding

- `kairis-aperture-mark.svg`
  - Abstract timing-window concept
  - Represents an opening, aperture, or opportune moment
  - Best if the brand should feel more premium and less explicitly technical

- `kairis-signal-gate.svg`
  - System-oriented signal crossing concept
  - Represents validation, gating, and controlled execution
  - Best if the brand should feel more operational and infrastructure-like

## Palette

- Background: `#111315`
- Light mark: `#E9E4D8`
- Green accent: `#708B7A`
- Brass accent: `#C8B27A`

## Selected Direction

The selected logo direction for Kairis is `kairis-threshold-k.svg`.

This is now the default source-of-truth concept for further refinement, favicon work, app icon derivation, and brand system expansion.

## Internal Module Use

The non-selected concepts are still intentionally retained:

- `kairis-aperture-mark.svg`
  - reserve for insight, analytics, reporting, or review surfaces

- `kairis-signal-gate.svg`
  - reserve for execution, controls, infrastructure, or orchestration surfaces
